# heart-generation-project
This project is an interactive web animation that creates a colorful trail of hearts following the user's cursor movements.
When you click and drag on the screen, the application continuously generates animated heart shapes that:
- Spawn at cursor position while the mouse button is held down
- Float upward with random horizontal drift
- Cycle through rainbow colors using CSS hue rotation
- Fade out gradually over 2 seconds before disappearing

The project uses vanilla JavaScript for interaction handling, SVG for heart graphics, and CSS animations for smooth visual effects. It features a full-screen black background that makes the colorful heart trails stand out prominently.
